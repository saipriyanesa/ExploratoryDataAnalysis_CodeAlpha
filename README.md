This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset using Python. The main objective is to understand the structure and quality of the data, identify trends and patterns, detect anomalies, and extract meaningful insights that support data-driven decision-making.

The analysis begins with examining the dataset's variables, data types, and missing values. Statistical summaries are generated to understand the distribution of sales and profit data. Various visualizations, including line charts, histograms, scatter plots, and heatmaps, are used to explore relationships between variables and uncover hidden patterns. The project also includes correlation analysis and outlier detection to ensure data reliability and accuracy.

Through EDA, valuable business insights are derived, such as identifying top-performing months, understanding profit trends, and evaluating the relationship between sales and profit.
